# DevOps — Контрольная работа  
**Агрегатор погодных данных для г. Волгограда (вариант 8.3)**

## 📌 Описание
Система собирает данные о погоде из двух источников:
- **Open-Meteo API** (реальный API без ключа)
- **Gismeteo** (эмуляция через статический JSON)

Данные нормализуются к единой схеме и сохраняются в PostgreSQL.  
Доступ предоставляется через:
- Веб-приложение (`/data`)
- pgAdmin (администрирование БД)
- Metabase (аналитика и дашборды)
